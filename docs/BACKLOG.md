# Бэклог — функционал на будущее

Здесь записаны идеи и функции, которые планируются, но пока не в приоритете.

## 1. Лимиты сообщений / монетизация
- ИИ-запросы стоят денег, нужно продумать ограничения
- Варианты: бесплатный лимит сообщений в день + платная подписка
- Детали пока не определены

## 2. Онбординг (первый запуск)
- Пошаговое заполнение профиля при регистрации (3-4 экрана)
- Рост, вес, цель, активность — по одному на экран

## 3. Экспорт данных
- Скачать таблицы в PDF или Excel
- Для врача/тренера

## 4. Уведомления
- Напоминания: "Ты сегодня ещё не записывал обед", "Не забудь выпить воду"
- Актуально когда приложение станет PWA/мобильным

## 5. Прогресс-система
- Прогресс-бар до цели по весу
- Серия дней (streak) — "5 дней подряд в норме калорий"
- Достижения/бейджи — "Первая неделя", "Месяц без срывов"

## 6. Графики в отчётах
- Визуальные графики вместо/вместе с таблицами в профиле
- Вес по неделям, калории по дням и т.д.
