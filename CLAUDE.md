## Проект

Calorie Counter — PWA-приложение для подсчёта калорий с ИИ-интеграцией. Язык интерфейса: русский.

## Стек технологий

- **Frontend:** React + Next.js (TypeScript)
- **Backend:** Java + Spring Boot
- **База данных:** PostgreSQL
- **ИИ:** Google Gemini API
- **Внешние API:** USDA, FatSecret (базы продуктов для точных данных БЖУ)
- **Будущее:** Capacitor для обёртки в мобильное приложение

## Архитектура

Два отдельных проекта в одном репозитории: frontend (Next.js) и backend (Spring Boot). Backend предоставляет REST API, frontend потребляет его. ИИ-запросы проксируются через backend.

### Ключевые сущности
- **Один чат с двумя режимами:** "Дневник" (записывает в таблицу) и "Вопрос" (просто ответ)
- **Таблицы питания** с каскадной группировкой: дни → недели → месяцы. Редактируются только через ИИ, пользователь не имеет прямого доступа к редактированию
- **Память ИИ (привычки):** ИИ запоминает предпочтения пользователя и учитывает при подсчёте (например "чай = чай + 2 ложки сахара")
- **Профиль:** рост, вес, возраст, пол, активность, цель (похудеть/набрать/поддерживать/мышечная масса)

## Команды разработки

| Действие | Команда |
|----------|---------|
| Запустить БД | `docker compose up -d` |
| Собрать backend | `cd backend && ./gradlew build` |
| Запустить backend | `cd backend && ./gradlew bootRun` |
| Собрать frontend | `cd frontend && npm run build` |
| Запустить frontend | `cd frontend && npm run dev` |

## Переменные окружения

Хранятся в `backend/.env` (не в git). Ключевые:
- `GEMINI_API_KEY` — ключ Google Gemini API
